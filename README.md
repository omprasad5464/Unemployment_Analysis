# Unemployment Analysis in India 📊

## 📌 Project Overview
Unemployment is measured by the unemployment rate, which is the number of people unemployed 
as a percentage of the total labour force. During Covid-19, India saw a sharp increase in 
unemployment. This project analyzes unemployment trends using Python and data science techniques.

## 🎯 Objectives
- Study unemployment rate patterns in India.
- Focus on the sharp rise during Covid-19.
- Visualize trends and insights using graphs.

## 🛠️ Tools & Libraries
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook (Google Colab)

## 📂 Dataset
- `Unemployment in India.csv`
- `Unemployment_Rate_upto_11_2020.csv`

## 📑 Steps Performed
1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Visualization of unemployment trends
4. Insights and conclusions

## 📈 Results
- Clear visualization of unemployment rate rise during Covid-19.
- Identified patterns in labour force participation.
